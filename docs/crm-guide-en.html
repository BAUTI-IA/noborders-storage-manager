›Before you start

The essentials to run the system day to day.

This is the system we use to manage pickups, deliveries, storage and billing. The key things:

  • Everything is live: if a teammate adds or changes something, you see it instantly — no need to reload the page.
  • It runs in the browser (computer or phone). Nothing to install.
  • Your work screen is Dispatching. That's where you'll spend 80% of your time.

Note: a few buttons and labels in the app are still in Spanish — this guide shows them exactly as they appear on screen, with the English meaning in parentheses.

4Create and edit a job

Tap “+ Nuevo job” (= New job), top right in Dispatching or Jobs.

Full

Pickup → storage → delivery. It's stored in between.

Direct

Pickup → delivery. Straight through, no storage.

Broker delivery

Delivery only: the goods are already here, you just deliver them.

What you fill in

  • Where it's stored — tick one or more units and/or warehouses (a job can occupy several places).
  • Job data — Job #, Customer, Type, Status, Broker, Driver, FADD, Volume (CF), Lot #, Sticker color.
  • Pickup and Delivery — date, address, city, state, zip and the balance to collect at each stage.
  • Storage billing — whether the client is charged monthly for storage (see section 7).
Edit later: tap the Job # to open the detail and click almost any field to change it on the spot (FADD, customer, driver, addresses, balances…). There's also an Editar (Edit) button for the full form.

Storage Units

Inside there are two sub-tabs:

  • Unidades (Units) — the physical lockers with their occupancy bar. If it's empty, tap “Set capacity”.
  • Jobs en unidades (Jobs by unit) — what's stored in each locker (one row per unit).

Add / import

“+ Unidad” (+ Unit) for a new unit. You can also Import from WhatsApp by pasting the message or uploading the chat .zip. The gate code copies with one click.

6Brokers & Drivers

🤝 Brokers

List of brokers with contact, phone and email. For each one you see how many jobs they have and the pending balance. You can add, edit or delete. When creating a job, pick the broker from the dropdown.

🧑‍✈️ Drivers

List of drivers. They populate automatically as you assign them in jobs. You see how many active and delivered jobs each one has.

8Golden rules & glossary

🎯 The 4 rules of the day

1. Check Dispatching and tackle the red/orange FADD first.
2. Make sure no pickup/delivery for today is left without a driver.
3. Always notify the driver with the job's 💬 WhatsApp button (it already includes all the data).
4. Advance the status as soon as each stage happens, so the board reflects reality.
